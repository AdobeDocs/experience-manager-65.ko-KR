# 관리 및 프로세스 용어 정보 {#about-administration-and-process-terminology}

>[!NOTE]
> 
> 사용자에게 관리자 콘솔에 액세스할 수 있는 관리자 권한이 있는지 확인하십시오.

Forms Workflow를 효과적으로 관리하려면 다음과 같은 용어와 요소 계층을 이해해야 합니다.

* *프로세스 카테고리*&#x200B;는 프로세스를 논리적 그룹으로 구성합니다.
* *프로세스*&#x200B;는 자동화된 비즈니스 프로세스를 나타냅니다. 구현된 각 비즈니스 프로세스에는 프로세스가 하나 이상 존재해야 합니다.
* *프로세스 다이어그램*&#x200B;은 자동화되고 있는 비즈니스 프로세스를 시각적으로 표현한 것입니다.
* *프로세스 인스턴스*&#x200B;는 사용자가 시작한 프로세스(예: 양식 작성 및 제출) 또는 자동화된 단계입니다. 한 프로세스에는 여러 인스턴스가 있을 수 있습니다.
* *운영*&#x200B;은 프로세스 다이어그램의 한 단계로, 워크벤치에서 프로세스 다이어그램의 아이콘으로 표시됩니다.
* *작업*&#x200B;은 사용자에게 할당된 작업입니다. 작업이 실행되고 운영이 실행되면 사용자 서비스는 사용자에게 작업을 할당합니다.

해당 용어에 대한 자세한 내용은 [워크벤치 도움말](https://www.adobe.com/go/learn_aemforms_workbench_63)을 참조하십시오.
