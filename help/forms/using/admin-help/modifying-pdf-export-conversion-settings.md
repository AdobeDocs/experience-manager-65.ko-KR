# PDF 내보내기 변환 설정 수정 {#modifying-the-pdf-export-conversion-settings}

PDF, EPS, DOC, TXT, RTF, XML, HTML 파일을 내보내기 위한 변환 설정을 수정하려면 아래 절차를 따릅니다. 기본적으로 PDF 파일에서는 Adobe Acrobat Professional 또는 Acrobat Standard에 구성된 다른 이름으로 저장 기본 설정을 사용합니다. 예를 들어 Acrobat에서 PDF 파일을 EPS로 변환할 때 다른 이름으로 저장 기본 설정을 사용하면 PDF 파일에서 한 페이지만 EPS로 변환됩니다.

>[!NOTE]
>
>한 파일 형식의 다른 이름으로 저장 설정을 수정하면 PDF Generator에서 내보낼 때 동일한 유형의 모든 변환에 해당 설정이 적용됩니다.

1. Acrobat에서 PDF 파일을 열고 파일 > 다른 이름으로 저장을 선택합니다.
1. 파일 형식 목록에서 적절한 형식을 선택합니다.
1. 설정을 클릭하고 필요에 따라 파일 형식을 설정합니다.
1. 확인을 클릭한 후 저장을 클릭하여 PDF 파일을 내보냅니다.
