# EMC Documentum®용 커넥터 사용자를 위한 백업 전략 {#backup-strategy-for-connector-for-emc-documentum-users}

EMC Documentum®용 커넥터가 설치되어 있는 경우 이 장에 명시된 지침 외에도 ECM 시스템이 설치된 컴퓨터를 백업(또는 복구)하는 작업을 백업 및 복구 전략에 포함해야 합니다. (ECM Documentum® 설명서 참조).

ECM 저장소를 사용하고 다음 작업을 수행하여 AEM Forms 환경을 백업하십시오.

* 이 문서에 설명된 지침을 따라 AEM Forms를 백업합니다.
* [EMC Documentum® 콘텐츠 서버 백업](/help/forms/using/admin-help/backing-recovering-emc-documentum-repository.md#back-up-the-emc-documentum-content-server)에 명시된 지침을 따라 ECM Documentum® 시스템을 백업하십시오.

ECM 저장소를 사용하고 다음 작업을 수행하여 AEM Forms 환경을 복원하십시오.

* [EMC Documentum® 콘텐츠 서버 복원](/help/forms/using/admin-help/backing-recovering-emc-documentum-repository.md#restore-the-emc-documentum-content-server)에 명시된 지침을 따라 해당 ECM 시스템을 복원하십시오.
* 이 문서에 설명된 지침을 따라 AEM Forms를 복원합니다.
