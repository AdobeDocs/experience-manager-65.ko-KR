# 사용자 또는 그룹 {#search-for-a-user-or-group} 검색

1. 관리 콘솔에서 설정 > 사용자 관리 > 사용자 및 그룹을 클릭합니다.
1. 검색 범위를 좁힐 정보를 지정하고 찾기를 클릭합니다. 검색 결과는 페이지 하단에 나열됩니다. 열 제목을 클릭하면 목록을 정렬할 수 있습니다. 최대 1000개의 결과가 반환됩니다.
