# 보안 설정 지정 {#specifying-security-settings}

양식을 사용하면 XML 입력의 외부 엔티티가 해결되는지 여부를 제어할 수 있습니다. 기본적으로 이러한 문제가 해결되지만 AEM 양식 시스템의 보안을 강화하도록 이 동작을 변경할 수 있습니다.

**외부 엔티티에 대한 참조가 들어 있는 XML 데이터 파일 처리 방지**

1. 관리 콘솔에서 서비스 > **[!UICONTROL 양식을 클릭합니다]**.
1. 외부 엔티티 해결 확인란을 선택 취소합니다.
1. **[!UICONTROL 저장]**&#x200B;을 클릭합니다.
