# PDF Generator 백업 제한 사항 {#pdf-generator-backup-limitations}

PDF Generator이 파일을 변환하는 데 사용하는 임시 디렉터리는 백업할 수 없습니다. 서비스가 제대로 복원되더라도 PDF Generator이 설정된 간격으로 임시 디렉터리의 내용을 검토하고 지우므로 데이터가 손실될 수 있습니다.
