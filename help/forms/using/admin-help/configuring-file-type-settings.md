# 파일 유형 설정 구성 {#configuring-file-type-settings}

>[!NOTE]
> 
> 사용자에게 관리자 콘솔에 액세스할 수 있는 관리자 권한이 있는지 확인하십시오.

PDF Generator에서는 지원되는 파일 유형에 대한 애플리케이션 설정을 지정할 수 있습니다. Windows에서는 지원되는 각 파일 유형에 대한 애플리케이션 설정을 지정할 수 있습니다. UNIX와 Linux에서는 HTML-PDF와 OpenOffice에 대한 애플리케이션 설정을 지정할 수 있습니다.

파일 유형 설정 페이지에서 다음 작업을 수행할 수 있습니다.

* [파일 유형 설정 만들기 또는 편집](#create-or-edit-file-type-settings)
* 기본적으로 사용할 파일 유형 설정 지정([PDF Generator 구성 파일 가져오기 및 내보내기](/help/forms/using/admin-help/importing-exporting-pdf-generator-configuration.md) 참조)
* [기본 설정 변경](/help/forms/using/admin-help/configuring-file-type-settings.md#change-the-default-settings)
* [PDF/A 지원 활성화](/help/forms/using/admin-help/enable-pdf-a-support.md)
* [파일 유형 설정 삭제](/help/forms/using/admin-help/enable-pdf-a-support.md)

>[!NOTE]
>
>HTML을 PDF로 변환하는 Acrobat, Microsoft PowerPoint, Microsoft Word, Microsoft Excel과 같은 대체 변환기에서는 파일 유형 설정을 사용할 수 없습니다.

## 파일 유형 설정 만들기 또는 편집 {#create-or-edit-file-type-settings}

파일 유형 설정을 만들거나 편집하여 애플리케이션에서 지원되는 파일 유형을 변환하는 작업을 처리하는 방법을 지정합니다. Windows에서는 지원되는 각 파일 유형에 대한 애플리케이션 설정을 지정할 수 있습니다. UNIX와 Linux에서는 HTML-PDF와 OpenOffice에 대한 애플리케이션 설정을 지정할 수 있습니다.

1. 관리 콘솔에서 **[!UICONTROL 서비스]** > **[!UICONTROL PDF Generator]** > **[!UICONTROL 파일 유형 설정]**&#x200B;을 클릭합니다.
1. 새로 만들기를 클릭하거나 설정 이름을 클릭합니다.
1. 파일 이름 확장자 상자에 이 애플리케이션에서 허용되는 파일 유형의 파일 이름 확장자를 쉼표로 구분하여 입력합니다. 확장자 앞에 마침표를 찍거나 확장자 사이에 공백을 넣지 마십시오. 기본값은 `bmp,gif,jpeg,jpg,tif,tiff,png`입니다.
1. (선택 사항) 그래픽이나 이미지에 포함된 텍스트에 대한 광학 코드 인식(OCR)을 사용하려면 OCR 사용을 선택하고 다음 옵션을 설정합니다.

**기본 OCR 언어:** OCR 엔진에서 문자를 식별하는 데 사용하는 언어입니다. 기본값은 영어(미국)입니다.

**PDF 출력 스타일:** 검색 가능한 이미지를 선택하면 페이지의 비트맵 이미지가 전경에 표시되고 스캔한 텍스트는 아래의 보이지 않는 레이어에 표시됩니다. 페이지 모양은 변하지 않지만, 텍스트를 선택하고 읽을 수 있게 됩니다. 서식 있는 텍스트 및 그래픽을 선택하면 인식된 텍스트, 글꼴, 사진 및 기타 그래픽 요소를 사용하여 원본 페이지를 재구성할 수 있습니다. 기본값은 검색 가능한 이미지(원본 그대로 유지)입니다.

**이미지 다운샘플링:** 색상, 회색 음영, 단색 이미지의 픽셀 수를 줄입니다. OCR이 완료된 후 스캔된 이미지의 다운샘플링이 수행됩니다. 기본값은 가장 낮음(600dpi)입니다. PDF 출력 스타일을 검색 가능한 이미지(원본 그대로 유지)로 설정하는 경우 이 옵션을 사용할 수 없습니다.

1. 다음 섹션에 필수 정보를 입력합니다.

## 기본 설정 변경 {#change-the-default-settings}

새로 생성된 소스에 적용되는 Adobe PDF 설정, 보안 설정 및 파일 유형 설정의 기본값을 변경할 수 있습니다. 기본값을 변경해도 기존 소스 설정에는 영향을 미치지 않습니다.

1. 관리 콘솔에서 **[!UICONTROL 서비스 > PDF Generator]**&#x200B;를 클릭합니다.
1. **[!UICONTROL Adobe PDF 설정]**, **[!UICONTROL 파일 유형 설정]** 또는 **[!UICONTROL 보안 설정]** 페이지에서 **[!UICONTROL 기본 설정 지정]**&#x200B;을 클릭합니다.
1. 원하는 기본 설정을 선택합니다. 다음 설정 중 하나 이상을 기본 설정 지정 페이지에서 사용할 수 있습니다.

   **[!UICONTROL Adobe PDF 설정]**: 원래 기본값은 표준(Acrobat 6)입니다.

   **[!UICONTROL 보안 설정]**: 원래 기본값은 보안 없음(Acrobat 5)입니다.

   **[!UICONTROL 파일 유형 설정]**: 원래 기본값은 표준입니다.

1. **[!UICONTROL 저장]**&#x200B;을 클릭합니다.

## 파일 유형 설정 삭제 {#delete-a-file-type-setting}

더 이상 사용되지 않는 파일 유형 설정을 삭제할 수 있습니다.

1. 관리 콘솔에서 **[!UICONTROL 서비스 > PDF Generator > 파일 유형 설정]**&#x200B;을 클릭합니다.
1. 삭제할 설정 옆에 있는 확인란을 선택합니다. 여러 소스를 선택할 수 있습니다. 옆에 확인란이 없는 설정은 항상 PDF Generator에 포함되어 있으며 삭제할 수 없습니다.
1. **[!UICONTROL 삭제]**&#x200B;를 클릭하고 삭제 확인 페이지에서 **[!UICONTROL 삭제]**&#x200B;를 클릭합니다.

## 이미지-PDF 설정 {#image-to-pdf-settings}

다음 옵션은 이미지 파일을 PDF로 변환하는 방법을 결정합니다. 해당 설정에 액세스하는 방법에 대한 지침은 [파일 유형 설정 만들기 또는 편집](configuring-file-type-settings.md#create-or-edit-file-type-settings)을 참조하십시오.

**파일 이름 확장자:** 변환 가능한 파일 이름 확장자를 쉼표로 구분한 목록입니다.

**대체 변환기 사용:** PDF Generator는 Java™ 또는 Acrobat을 사용하여 이미지 파일을 PDF로 변환할 수 있습니다. 이 옵션을 선택한 상태에서 변환이 실패하거나 지정된 시간 초과 제한에 도달하면 PDF Generator에서 대체 방법을 사용하여 변환을 시도합니다. 대체 방법도 실패하거나 지정된 시간 초과 제한에 도달하면 예외가 로그 파일에 기록됩니다.

>[!NOTE]
>
>JPEG 2000 파일은 Acrobat을 사용해서만 변환할 수 있습니다.

**OCR 사용:** PDF에 OCR(광학 문자 인식)을 적용할지 여부를 지정합니다. OCR 소프트웨어를 사용하면 PDF의 텍스트를 검색하고 수정하며 복사할 수 있습니다.

***참고&#x200B;**: OCR PDF(검색 가능한 PDF) 기능은 Microsoft Windows에서만 지원됩니다.*

**기본 OCR 언어:** OCR 엔진에서 문자를 식별하는 데 사용하는 언어를 지정합니다.

**PDF 출력 스타일:** 생성할 PDF 유형을 결정합니다. 모든 형식은 OCR과 글꼴 및 페이지 인식 기능을 텍스트 이미지에 적용하여 일반 텍스트로 변환합니다.

**검색 가능한 이미지:** 텍스트를 검색하고 선택할 수 있도록 합니다. 이 옵션을 선택하면 원본 이미지가 유지되고 필요에 따라 해당 이미지의 기울어짐을 보정한 후 그 위에 보이지 않는 텍스트 레이어가 추가됩니다. 이미지 다운샘플링 옵션은 이미지를 다운샘플링할지 여부와 다운샘플링 정도를 결정합니다.

**검색 가능한 이미지(원본 그대로 유지):** 텍스트를 검색하고 선택할 수 있도록 합니다. 이 옵션을 선택하면 원본 이미지가 유지되고 그 위에 보이지 않는 텍스트 레이어가 추가됩니다. 원본 이미지에 최대한 충실해야 하는 경우에 권장됩니다.

**ClearScan:** 원본에 매우 가까운 새로운 Type 3 글꼴을 합성하고 저해상도 사본을 사용하여 페이지 배경을 보존합니다.

**이미지 다운샘플링:** OCR이 완료된 후 색상, 회색 음영, 단색 이미지의 픽셀 수를 줄입니다. 적용할 다운샘플링 정도를 선택합니다. 숫자가 더 높은 옵션을 사용하면 다운샘플링 정도가 줄어들어 해상도가 더 높은 PDF가 생성됩니다.

## Adobe PDF 내보내기 설정(Windows만 해당) {#adobe-pdf-export-settings-windows-only}

Adobe PDF 내보내기 설정 섹션의 내보내기 파일 유형 설정은 PDF 파일을 다른 형식으로 변환하는 데 사용됩니다. 기본값은 CSS(Cascading Style Sheets) 1.0(*.htm, *.html)이 적용된 HTML 4.01입니다.

해당 설정에 액세스하는 방법에 대한 지침은 [파일 유형 설정 만들기 또는 편집](configuring-file-type-settings.md#create-or-edit-file-type-settings)을 참조하십시오.

## HTML-PDF 설정 {#html-to-pdf-settings}

다음 옵션은 HTML 파일을 PDF로 변환하는 방법을 결정합니다. 해당 옵션에 액세스하는 방법에 대한 지침은 [파일 유형 설정 만들기 또는 편집](configuring-file-type-settings.md#create-or-edit-file-type-settings)을 참조하십시오.

**대체 변환기 사용:** PDF Generator는 Java™ 또는 Acrobat을 사용하여 HTML 파일을 PDF로 변환할 수 있습니다. 이 옵션을 선택한 상태에서 변환이 실패하거나 지정된 시간 초과 제한에 도달하면 PDF Generator에서 대체 방법을 사용하여 변환을 시도합니다. 대체 방법도 실패하거나 지정된 시간 초과 제한에 도달하면 예외가 로그 파일에 기록됩니다.

**기본 인코딩:** 운영 체제 및 알파벳 메뉴에서 파일 텍스트의 입력 인코딩을 설정합니다. HTML 소스 파일에 인코딩 유형이 지정되지 않은 경우에만 기본 인코딩 옵션에 표시된 선택 항목을 사용합니다.

**선택한 인코딩 강제 적용:** HTML 소스 파일에 지정된 모든 인코딩을 무시하고 기본 인코딩 옵션에 표시된 선택 항목을 사용합니다.

### 스파이더링 설정 {#spidering-settings}

*스파이더링*&#x200B;은 웹 페이지에서 다른 웹 페이지로 연결되는 링크를 스캔합니다. 다른 웹 페이지로 연결되는 링크가 발견되면 대상 페이지를 가져와서 생성된 PDF 문서에 포함합니다. 다음 옵션을 활성화하면 가져와서 PDF로 변환할 수준 수를 설정할 수 있습니다.

**X수준만 가져오기:** 기본 페이지 URL에서 지정된 수준까지 페이지를 스파이더링하고 변환합니다. 값 1은 제공된 URL만 변환합니다.

**전체 사이트 가져오기:** 제공된 URL부터 시작하여 전체 사이트를 변환합니다.

**동일한 경로 유지:** 기본 URL과 동일한 상대 경로에 있지 않은 페이지를 가리키는 모든 링크가 스파이더링 중에 변환되지 않습니다.

**동일한 서버 유지:** 다른 서버의 페이지를 가리키는 모든 링크가 스파이더링 중에 변환되지 않습니다. 지정된 URL과 동일한 서버를 가리키는 링크만 변환됩니다.

### 페이지 변환 설정 {#page-conversion-settings}

다음 옵션을 활성화하면 HTML 페이지를 변환하는 방법을 지정할 수 있습니다. 페이지 크기에 따라 폭, 높이, 여백 값이 적절히 조정됩니다.

**페이지 크기:** 사용자 정의를 선택하고 폭과 높이를 지정하거나 미리 정의된 크기를 선택합니다.

**방향:** 변환된 PDF 문서의 세로 방향 또는 가로 방향을 선택합니다.

**여백:** 생성된 PDF 문서의 여백(위쪽, 아래쪽, 왼쪽, 오른쪽)을 지정합니다.

**PDF에 책갈피 추가:** PDF 문서에 책갈피를 추가합니다.

**태그가 지정된 PDF 활성화:** PDF 문서에 태그를 임베드합니다.

**초기 보기 설정 지정:** 문서 옵션, 창 옵션, 사용자 인터페이스 옵션을 구성할 수 있습니다. 해당 설정은 콘텐츠가 처음에 표시되는 방법을 결정합니다.

### 문서 옵션 {#document-options}

다음 옵션을 활성화하면 콘텐츠를 표시하는 방법, PDF 문서의 페이지를 표시하는 방법, 확대 배율 수준을 설정하는 방법을 지정할 수 있습니다.

**표시:** PDF 문서를 열 때 Acrobat에서 열 창을 선택합니다.

**페이지 레이아웃:** PDF 문서의 페이지 레이아웃 유형을 선택합니다.

**확대:** PDF 문서를 처음 볼 때 사용할 사전 설정된 확대 배율을 선택하거나 사용자 정의 값을 선택합니다. 기본 설정을 선택하면 기본 Acrobat 확대 배율이 사용됩니다.

**페이지 번호로 열기:** PDF가 열리는 페이지 번호를 지정합니다.

### 창 옵션 {#window-options}

다음 옵션을 활성화하면 창의 크기와 표시 방식을 지정할 수 있습니다.

**창 크기를 초기 페이지로 조정:** Acrobat 창의 크기를 초기 페이지 크기로 조정합니다.

**화면 중앙에 창 배치:** 화면 중앙에서 창을 엽니다.

**전체 화면 모드로 열기:** 창을 전체 화면 모드로 엽니다.

**표시:** 창에 문서 제목이나 파일 이름을 표시합니다.

### 사용자 인터페이스 옵션 {#user-interface-options}

다음 옵션을 활성화하면 창 모양을 지정할 수 있습니다.

**메뉴 모음 숨기기:** PDF 문서에서 메뉴 모음을 숨깁니다.

**도구 모음 숨기기:** PDF 문서에서 도구 모음을 숨깁니다.

**창 컨트롤 숨기기:** PDF 문서에서 창 컨트롤을 숨깁니다.

## 플래시 비디오-PDF 설정 {#flash-videos-to-pdf-settings}

PDF Generator는 Adobe Flash용 비디오(SWF 또는 FLV 파일)를 제출하고 Adobe Flash용 비디오가 임베드된 PDF 파일을 만드는 기능을 지원합니다. 이 변환 작업은 Forms 서버에 Adobe Flash Player를 설치하지 않아도 수행할 수 있습니다. 해당 옵션에 액세스하는 방법에 대한 지침은 [파일 유형 설정 만들기 또는 편집](configuring-file-type-settings.md#create-or-edit-file-type-settings)을 참조하십시오.

**파일 이름 확장자:** 변환 가능한 파일 이름 확장자를 쉼표로 구분한 목록입니다.

## XPS-PDF 설정 {#xps-to-pdf-settings}

Windows 인쇄기에서는 XPS(XML Paper Specification)가 사용됩니다. XPS는 Microsoft 형식이며 모든 Microsoft Office 애플리케이션에서 만들 수 있습니다. AEM Forms는 XPS 파일을 PDF로 변환하는 기능을 제공합니다.

**파일 이름 확장자:** 변환 가능한 모든 XPS 파일 이름 확장자를 쉼표로 구분한 목록입니다. 현재는 .xps라는 한 가지 형식만 있습니다.

## PDF 최적화 설정 {#pdf-optimizer-settings}

PDF Generator는 PDF 파일의 크기를 줄이는 기능을 지원합니다. 이러한 설정을 모두 사용할지 아니면 일부만 사용할지는 파일을 사용하는 방법과 파일에 반드시 필요한 필수 속성에 따라 달라집니다. 대부분의 경우 기본 설정은 효율성을 극대화하는 데 적합합니다. 즉, 임베드된 글꼴을 제거하고, 이미지를 압축하고, 더 이상 필요하지 않은 파일에서 항목을 제거하여 공간을 절약합니다.

>[!NOTE]
>
>디지털 서명된 문서를 최적화하면 디지털 서명이 제거되고 무효화됩니다.

해당 설정에 액세스하는 방법에 대한 지침은 [파일 유형 설정 만들기 또는 편집](configuring-file-type-settings.md#create-or-edit-file-type-settings)을 참조하십시오.

**대상 PDF 버전:** PDF와 호환되는 Acrobat 버전을 지정합니다.

### 글꼴 {#fonts}

1. **글꼴**&#x200B;을 선택합니다.
1. 다음 옵션 중 하나를 선택하십시오.

   **모든 글꼴 임베드 취소:** 임베드된 모든 글꼴의 임베드를 취소합니다.

   **모든 글꼴 임베드 취소 안 함:** 모든 글꼴의 임베드를 취소하지 않습니다.

   **일부 글꼴 임베드 취소:** 지정된 글꼴만 임베드를 취소합니다. 다음 단계를 따라 임베드를 취소할 글꼴을 지정하십시오.

   * 필요한 경우 **글꼴 소스** 드롭다운 메뉴에서 다른 글꼴 디렉터리를 선택합니다. 이 드롭다운 메뉴에는 **홈 > 설정 > 핵심 시스템 > 핵심 구성**&#x200B;에 지정된 글꼴 디렉터리가 나열되어 있습니다.
   * **사용 가능한 글꼴** 목록에서 글꼴을 하나 이상 선택하고 **추가**&#x200B;를 클릭합니다. 해당 글꼴은 **임베드를 취소할 글꼴** 목록에 추가됩니다.
   * Forms 서버에 없는 일부 글꼴의 임베드를 취소하려면 **임베드를 취소할 글꼴 추가** 상자에 해당 글꼴의 이름을 입력합니다. **추가**&#x200B;를 클릭합니다.

   >[!NOTE]
   >
   >*문서에 하위 집합이 임베드된 일부 글꼴의 임베드를 취소하려면 글꼴 이름 앞에 + 기호를 붙이십시오. 예를 들어 &#39;+Helvetica&#39;입니다.*

1. 임베드된 글꼴에서 사용 중인 하위 집합만 임베드하려면 **임베드된 모든 글꼴의 하위 집합**&#x200B;을 선택합니다.

   >[!NOTE]
   >
   >*이 옵션을&#x200B;**일부 글꼴 임베드 취소**와 함께 사용하는 경우&#x200B;**임베드를 취소할 글꼴 추가**목록에 있는 글꼴은 여전히 완전하게 임베드 취소됩니다.*

   >[!NOTE]
   >
   >*글꼴 하위 설정은 글꼴의 일부만 임베드하는 기술입니다. 글꼴 하위 집합에는 문서에서 사용되는 문자만 포함됩니다.*

### 투명도 {#transparency}

PDF 문서에 투명도가 적용된 아트워크가 포함되어 있는 경우 PDF 최적화 설정을 사용하여 투명도를 평면화하고 파일 크기를 줄일 수 있습니다.

>[!NOTE]
>
>대상 PDF 버전으로 Acrobat 4. 0 이상을 선택하면 투명한 오브젝트가 모두 평면화됩니다. 다른 대상 PDF 버전의 경우 투명도가 지원되며 투명도 설정을 구성할 수 있습니다.

PDF 문서를 최적화하는 동안 투명도 설정을 구성하려면 **투명도**&#x200B;를 선택합니다.

**투명도 수준** 보존할 벡터 정보의 양을 지정합니다. 설정값이 높을수록 벡터 오브젝트가 더 많이 보존되고 설정값이 낮을수록 벡터 오브젝트가 더 많이 래스터화됩니다. 중간 설정값은 단순한 영역을 벡터 형식으로 보존하고 복잡한 영역을 래스터화합니다. 모든 아트워크를 래스터화하려면 가장 낮은 설정값을 선택합니다.

>[!NOTE]
>
>발생하는 래스터화의 양은 페이지의 복잡성과 겹치는 오브젝트 유형에 따라 달라집니다.

**라인 아트 및 텍스트** 이미지, 벡터 아트워크, 텍스트, 그라데이션 등 모든 오브젝트가 래스터화되는 해상도입니다. 지원되는 값은 인치당 픽셀(ppi) 기준 1~9,600ppi입니다.

>[!NOTE]
>
>특히 세리프 글꼴이나 작은 포인트 크기의 문자에서 고품질 래스터화를 제공하려면 라인 아트 및 텍스트 해상도를 일반적으로 600~1,200ppi로 설정해야 합니다.

**그라데이션 및 메시** 그라데이션과 메시가 래스터화되는 해상도입니다. 지원되는 값은 1~1,200ppi입니다.

>[!NOTE]
>
>그라데이션 및 메시 해상도는 일반적으로 150~300ppi로 설정해야 합니다. 해상도가 높아져도 그라데이션, 그림자, 페더 품질은 향상되지 않고 인쇄 시간과 파일 크기가 늘어나기 때문입니다.

**모든 텍스트를 윤곽선으로 변환** 모든 문자 오브젝트(포인트 문자, 영역 문자, 경로 문자)를 윤곽선으로 변환하고 투명도가 포함된 페이지에서 모든 문자 글리프 정보를 삭제합니다. 이 옵션을 사용하면 평면화하는 동안 텍스트 폭이 일관되게 유지됩니다. 이 옵션을 활성화하면 Acrobat에서 보거나 저해상도 데스크탑 프린터로 인쇄할 때 작은 글꼴이 약간 두껍게 표시됩니다. 하지만 고해상도 프린터나 이미지세터에 인쇄된 문자의 품질에는 영향을 미치지 않습니다.

**모든 획을 윤곽선으로 변환** 투명도가 포함된 페이지에서 모든 획을 단순한 채워진 경로로 변환합니다. 이 옵션을 사용하면 평면화하는 동안 획 폭이 일관되게 유지됩니다. 이 옵션을 활성화하면 얇은 획이 약간 두껍게 보이고 평면화 성능이 저하될 수 있습니다.

**복잡한 영역 잘라 내기** 벡터 아트워크와 래스터화된 아트워크 사이의 경계가 오브젝트 경로를 따라가도록 보장합니다. 이 옵션은 오브젝트의 일부로 인해 발생하는 스티칭 아티팩트를 줄여 줍니다.

다음 표는 일반적인 프린터 유형과 dpi로 측정한 해상도, 인치당 선 수(lpi)로 측정한 기본 스크린 선수, 인치당 픽셀 수(ppi)로 측정한 이미지의 리샘플링 해상도를 보여 줍니다. 예를 들어 600dpi 레이저 프린터로 인쇄하는 경우 이미지를 리샘플링할 해상도로 170을 입력합니다.

**이미지** 이미지를 선택하여 색상, 회색 음영, 단색 이미지의 압축 및 리샘플링 옵션을 지정합니다. 파일 크기와 이미지 품질 간의 적절한 균형을 찾으려면 해당 옵션을 실험해 보는 것이 좋습니다. 색상 및 회색 음영 이미지의 해상도 설정은 파일이 인쇄될 선 스크린 선수의 1.5~2배여야 합니다. 단색 이미지의 해상도는 출력 장치와 동일해야 하지만, 단색 이미지를 1,500dpi보다 높은 해상도로 저장하면 이미지 품질은 눈에 띄게 향상되지 않으면서 파일 크기만 늘어납니다. 지도 등 확대되는 이미지에는 더 높은 해상도가 필요할 수 있습니다.

>[!NOTE]
>
>단색 이미지를 리샘플링하면 이미지가 표시되지 않는 등 예기치 않은 결과가 발생할 수 있습니다. 이러한 현상이 발생하면 리샘플링을 끄고 파일을 다시 변환하십시오. 이 문제는 서브샘플링에서 발생할 가능성이 가장 높고 바이큐빅 다운샘플링에서는 발생할 가능성이 가장 낮습니다.

#### 오브젝트 삭제 {#discard-objects}

* **오브젝트 삭제**&#x200B;를 선택하면 PDF에서 제거할 오브젝트를 지정하고 CAD 도면의 곡선을 최적화할 수 있습니다.
* **모든 양식 제출, 가져오기 및 재설정 작업 취소**: 양식 데이터를 제출하거나 가져오는 것과 관련된 모든 작업을 비활성화하고 양식 필드를 재설정합니다. 이 옵션은 작업이 연결된 양식 오브젝트를 유지합니다.
* **모든 JavaScript 작업 취소**: JavaScript를 사용하는 모든 작업을 PDF에서 제거합니다.
* **임베드된 페이지 썸네일 삭제**: 임베드된 페이지 썸네일을 제거합니다. 이 옵션은 페이지 버튼을 클릭한 후 페이지 썸네일을 그리는 데 시간이 오래 걸릴 수 있는 대용량 문서에 유용합니다.
* **부드러운 선을 곡선으로 변환**: CAD 도면에서 곡선을 만드는 데 사용되는 제어점의 수를 줄여 PDF 파일의 크기를 줄이고 화면 렌더링 속도를 높입니다.
* **임베드된 인쇄 설정 취소**: 문서에서 페이지 크기 조정 및 양면 인쇄 모드와 같이 임베드된 인쇄 설정을 제거합니다.
* **책갈피 삭제**: 문서에서 모든 책갈피를 제거합니다.
* **양식 필드 평면화**: 모양을 변경하지 않고도 양식 필드를 사용할 수 없게 만듭니다. 양식 데이터는 페이지와 병합되어 페이지 콘텐츠가 됩니다.
* **모든 대체 이미지 삭제**: 화면에 표시될 버전을 제외한 모든 이미지 버전을 제거합니다. 일부 PDF에는 다양한 목적에 맞게 동일한 이미지의 여러 버전이 포함되어 있습니다. 예를 들어 저해상도 화면 보기와 고해상도 인쇄가 있습니다.
* **문서 태그 삭제**: 문서에서 태그를 제거하면 텍스트의 접근성 및 재배치 기능도 제거됩니다.
* **이미지 조각 감지 및 병합**: 얇은 조각으로 조각난 이미지 또는 마스크를 찾아 해당 조각을 단일 이미지 또는 마스크로 병합하려고 합니다.
* **임베드된 검색 색인 삭제**: 임베드된 검색 색인을 제거하여 파일 크기를 줄입니다.

#### 사용자 데이터 삭제 {#discard-user-data}

**사용자 데이터 삭제**&#x200B;를 선택하면 배포하고 싶지 않거나 다른 사용자와 공유하고 싶지 않은 개인 정보를 제거할 수 있습니다.

* **모든 주석, 양식 및 멀티미디어 삭제**: PDF에서 모든 주석, 양식, 양식 필드, 멀티미디어를 제거합니다.
* **모든 오브젝트 데이터 삭제**: PDF에서 모든 오브젝트를 제거합니다.
* **외부 상호 참조 삭제**: 다른 문서로 연결되는 링크를 제거합니다. PDF 내의 다른 위치로 이동하는 링크는 제거되지 않습니다.
* **숨겨진 레이어 콘텐츠 삭제 및 표시되는 레이어 평면화**: 파일 크기를 줄입니다. 최적화된 문서는 원본 PDF와 유사하지만, 레이어 정보는 포함하고 있지 않습니다.
* **문서 정보 및 메타데이터 삭제**: 문서 정보 사전과 모든 메타데이터 스트림의 정보를 제거합니다. (PDF 사본으로 메타데이터 스트림을 복원하려면 다른 이름으로 저장 명령을 사용하십시오.)
* **첨부 파일 삭제**: PDF에 주석으로 추가된 첨부 파일을 포함하여 모든 첨부 파일을 제거합니다. (PDF 최적화는 첨부 파일을 최적화하지 않습니다.)
* **다른 애플리케이션의 개인 데이터 삭제**: 문서를 만든 애플리케이션에만 유용한 정보를 PDF 문서에서 제거합니다. 이 설정은 PDF 기능에는 영향을 미치지 않지만, 파일 크기가 줄어듭니다.

### 정리 {#clean-up}

**정리**를 선택하면 문서에서 불필요한 항목을 제거할 수 있습니다.
이러한 항목에는 문서 용도와 다르게 더 이상 사용되지 않거나 불필요한 요소가 포함되어 있습니다. 특정 요소를 제거하면 PDF 기능에 심각한 영향을 미칠 수 있습니다. 기본적으로 기능에 영향을 미치지 않는 요소만 선택됩니다. 다른 옵션을 제거했을 때 어떤 영향을 미치는지 확실히 알 수 없다면 기본 선택 항목을 사용하십시오.

**압축**

드롭다운 메뉴에서 다음과 같은 Flate 압축 옵션 중 하나를 선택합니다.

* 전체 파일 압축
* 문서 구조 압축
* 압축 제거
* 압축을 그대로 유지

**Flate을 사용하여 인코딩되지 않은 스트림 인코딩**: 인코딩되지 않은 모든 스트림에 Flate 압축을 적용합니다.

**잘못된 책갈피 삭제**: 삭제된 문서의 페이지를 가리키는 책갈피를 제거합니다.

**참조되지 않은 명명된 대상 삭제**: PDF 문서 내부에서 참조되지 않는 명명된 대상을 제거합니다. 이 옵션은 다른 PDF 파일이나 웹 사이트의 링크를 확인하지 않습니다.

**빠른 웹 보기를 위한 PDF 최적화**: 웹 서버에서 한 번에 페이지 단위로 다운로드(바이트 제공)할 수 있도록 PDF 문서를 재구성합니다.

**LZW 인코딩을 사용하는 스트림에서 대신 Flate 사용**: LZW 인코딩을 사용하는 모든 콘텐츠 스트림과 이미지에 Flate 압축을 적용합니다.

**잘못된 링크 삭제**: 잘못된 대상으로 이동하는 링크를 제거합니다.

**페이지 콘텐츠 최적화**: 모든 줄 바꿈 문자를 공백 문자로 변환하여 Flate 압축을 향상합니다.

## Microsoft Excel 설정(Windows만 해당) {#microsoft-excel-settings-windows-only}

다음 옵션은 Microsoft Excel 파일을 변환하는 방법을 결정합니다. 해당 옵션에 액세스하는 방법에 대한 지침은 [파일 유형 설정 만들기 또는 편집](#create-or-edit-file-type-settings)을 참조하십시오.

**OpenOffice를 대체 변환기로 사용**: 이 옵션을 선택한 상태에서 Microsoft Excel을 사용한 변환이 실패하거나 지정된 시간 초과 제한에 도달하면 PDF Generator에서 OpenOffice를 사용하여 변환을 시도합니다. OpenOffice를 사용한 변환이 실패하거나 지정된 시간 초과 제한에 도달하면 예외가 로그 파일에 기록됩니다.

**파일 이름 확장자**: 이 애플리케이션에서 허용되는 파일 유형의 파일 이름 확장자를 쉼표로 구분하여 지정합니다. 기본값은 `xls,xlsx`입니다. 확장자 앞에 마침표를 찍거나 확장자 사이에 공백을 넣지 마십시오.

**PDF/A-1a 호환 파일 만들기**: PDF/A-1b:2005 RGB Adobe PDF 설정을 사용하도록 강제 적용합니다.

**Adobe PDF에 책갈피 추가**: Excel 워크시트 이름을 책갈피로 변환합니다. 이 옵션은 기본적으로 선택되어 있습니다.

**워크시트를 한 페이지에 맞추기**: 텍스트 크기를 줄여서 워크시트를 한 페이지에 맞춥니다.

**전체 통합 문서 변환**: Excel 파일에 있는 모든 워크시트를 변환합니다. 이 옵션을 선택하지 않으면 현재 페이지만 변환됩니다.

**매크로 자동 실행**: 문서를 변환하기 전에 Excel 문서에 있는 모든 매크로(예: 현재 시간을 삽입하는 매크로)를 실행합니다.

**문서 정보 변환**: 소스 파일의 문서 정보를 기반으로 PDF 문서 속성을 추가합니다. 여기에는 문서 제목, 작성자, 주제, 키워드와 같은 정보가 포함됩니다.

**Adobe PDF에 링크 추가**: 소스 파일의 하이퍼링크를 PDF 문서의 하이퍼링크로 변환합니다.

**Adobe PDF에 소스 파일 첨부**: 이 옵션을 선택하면 원본 Excel 스프레드시트가 생성된 PDF 문서 내부에 첨부 파일로 삽입됩니다.

**태그가 지정된 Adobe PDF로 접근성 및 재배치 활성화**: PDF 문서 내부에 태그를 임베드하여 접근성 및 재배치를 활성화합니다.

**로드할 Excel 추가 기능 목록**: 기본적으로 (보안상의 이유로) Excel 파일을 PDF로 변환할 때 Excel 추가 기능은 실행되지 않습니다. 변환 중에 특정 Excel 추가 기능을 실행할 수 있도록 하려면 추가 기능 이름을 쉼표로 구분한 목록을 제공하십시오.

**변환할 워크시트 목록**: 이 상자가 비어 있으면 Excel 스프레드시트의 모든 워크시트가 생성된 PDF에 포함됩니다. 워크시트의 하위 집합을 선택적으로 변환하려면 워크시트 이름을 쉼표로 구분한 목록을 제공하십시오.

## OpenOffice 설정 {#openoffice-settings}

다음 옵션은 OpenOffice 파일을 변환하는 방법을 결정합니다. 해당 옵션에 액세스하는 방법에 대한 지침은 [파일 유형 설정 만들기 또는 편집](/help/forms/using/admin-help/configuring-file-type-settings.md#create-or-edit-file-type-settings)을 참조하십시오.

**PDFMaker를 대체 변환기로 사용**: 이 옵션을 선택한 상태에서 OpenOffice를 사용한 변환이 실패하거나 지정된 시간 초과 제한에 도달하면 PDF Generator에서 PDFMaker를 사용하여 변환을 시도합니다. PDFMaker를 사용한 변환이 실패하거나 지정된 시간 초과 제한에 도달하면 예외가 로그 파일에 기록됩니다.

**파일 이름 확장자**: 이 애플리케이션에서 허용되는 파일 유형의 파일 이름 확장자를 쉼표로 구분하여 지정합니다. 기본값은 `odt,odp,ods,odg,odf,sxw,sxi,sxd`입니다. 확장자 앞에 마침표를 찍거나 확장자 사이에 공백을 넣지 마십시오.

**범위**: 모든 페이지를 변환하거나 특정 페이지 또는 페이지 범위를 지정합니다. 페이지 범위가 정의되지 않으면 모든 페이지가 변환됩니다. 여러 페이지를 내보내려면 3-6 형식을 사용합니다. 단일 페이지를 내보내려면 7;9;11 형식을 사용합니다. 3-6;8;10;12와 같은 형식을 사용하면 여러 페이지 범위와 단일 페이지를 조합하여 내보낼 수 있습니다.

**페이지 방향**: 일반 텍스트 파일의 경우에만 변환된 PDF 문서에 대해 세로 또는 가로를 선택합니다.

**이미지**: 이미지를 변환하는 방법을 구성합니다. 미리 보기가 임베드된 EPS 이미지는 미리 보기로만 내보내집니다. 미리 보기가 임베드되지 않은 EPS 이미지는 빈 자리 표시자로 내보내집니다. 이미지를 무손실 압축하면 모든 픽셀이 보존됩니다. 이미지를 JPEG 압축 방식으로 고품질 수준에서 압축하면 거의 모든 픽셀이 보존됩니다. 저품질 수준에서 압축하면 일부 픽셀이 손실되고 아티팩트가 생기지만 파일 크기는 줄어듭니다.

**일반**: 태그가 지정된 PDF를 변환하거나 Writer 및 FormCalc 문서 메모, Impress 슬라이드 전환 효과 또는 빈 페이지를 PDF로 내보내려면 옵션을 활성화합니다. 태그를 내보내면 파일 크기가 엄청나게 커질 수 있습니다. 내보내지는 일부 태그로는 목차, 하이퍼링크, 컨트롤이 있습니다.

양식을 제출하는 방법도 지정할 수 있습니다. 옵션은 XML, FDF, PDF, HTML입니다. 이 설정은 문서에서 설정한 컨트롤의 URL 속성을 재정의합니다. PDF 문서에 대해 공통 설정을 하나만 선택할 수 있습니다.

* PDF(문서 전체 전송)
* FDF(컨트롤 콘텐츠 전송)
* HTML
* XML

**태그가 지정된 PDF**: OpenOffice 문서에서 태그가 지정된 PDF를 만들 수 있습니다. 태그가 지정된 PDF에는 문서 콘테츠 구조에 대한 정보가 포함되어 있습니다. 다양한 화면이 탑재된 장치에서 문서를 표시하거나 스크린 리더 소프트웨어를 사용할 때 도움이 됩니다. 또한 접근성 소프트웨어에서 PDF 문서에 대해 여러 유용한 작업(예: PDF 문서 콘텐츠를 소리 내어 읽는 작업)을 수행하는 데 도움이 됩니다.

**메모 내보내기**: OpenOffice 문서의 메모를 생성된 PDF 문서의 메모로 변환합니다.

**전환 효과 사용**: OpenOffice 프레젠테이션의 슬라이드 전환 효과를 해당 PDF 전환 효과로 변환합니다.

**형식으로 양식 제출**: PDF 문서 사용자가 작성하고 인쇄할 수 있는 PDF 양식을 만듭니다.

**자동으로 삽입된 빈 페이지 내보내기**: 이 옵션을 선택하면 자동으로 삽입된 빈 페이지가 생성된 PDF 문서에 포함됩니다. 이 기능은 PDF 문서를 양면으로 인쇄하는 경우에 유용합니다. 예를 들어 각 장의 첫 페이지가 항상 홀수 페이지에서 시작하도록 책을 구성할 수 있습니다. 이전 장이 홀수 페이지에서 끝나면 OpenOffice는 빈 짝수 페이지를 삽입합니다. 이 옵션은 생성된 PDF에 짝수 페이지를 포함할지 여부를 제어합니다.
