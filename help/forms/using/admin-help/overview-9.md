# Forms Workflow 개요 {#overview-of-forms-workflow}

Forms Workflow를 사용하면 사용자가 인간 중심 프로세스를 설계, 관리 및 최적화할 수 있습니다. 사용자는 Forms Workflow를 사용하여 시스템과 사람을 연결하는 자동화된 비즈니스 프로세스 애플리케이션을 관리할 수도 있습니다.
