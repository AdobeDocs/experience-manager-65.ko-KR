# Forms 워크플로 개요 {#overview-of-forms-workflow}

Forms 워크플로를 통해 사용자는 사람 중심의 프로세스를 디자인, 관리 및 최적화할 수 있습니다. 양식 워크플로우를 사용하여 시스템과 직원을 연결하는 자동화된 비즈니스 프로세스 애플리케이션을 관리할 수도 있습니다.
