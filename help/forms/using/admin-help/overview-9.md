# Forms 워크플로우 개요 {#overview-of-forms-workflow}

Forms 워크플로우를 통해 사용자는 인간 중심 프로세스를 디자인, 관리 및 최적화할 수 있습니다. 또한 Forms Workflow를 사용하여 시스템과 사용자를 연결하는 자동화된 비즈니스 프로세스 애플리케이션을 관리할 수 있습니다.
