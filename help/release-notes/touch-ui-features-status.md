# Touch UI 기능 상태 {#touch-ui-feature-status}

AEM 6.4 이상 [클래식 UI는 더 이상 사용되지 않습니다](../release-notes/deprecated-removed-features.md). Adobe은 클래식 UI를 추가로 개선하지 않으며 터치 지원 UI에서 사용할 수 있는 강력한 새 기능을 활용할 것을 권장합니다.

버전 6.0부터 AEM에는 다음과 같이 정렬된 &quot;터치 지원 UI&quot;(단순히 &quot;Touch UI&quot;라고 함)라는 새 사용자 인터페이스가 도입되었습니다 [!DNL Adobe Experience Cloud] 및 를 전반적인 Adobe 사용자 인터페이스 지침에 따라 생성할 수 있습니다. 기능 패리티에 도달하면 레거시 &quot;클래식 UI&quot;라고 하는 데스크톱 기반 인터페이스를 사용하는 AEM의 표준 UI가 됩니다.

대부분의 기능이 터치 지원 UI에 표시되지만 아직 완성되지 않아 향후 릴리스에서 추가될 기능도 있습니다.

다음 목록은 AEM 6.5에 구현된 기능의 현재 상태를 보여줍니다.

AEM 6.5로 업그레이드하는 고객을 위한 권장 사항은 [고객을 위한 사용자 인터페이스 권장 사항](/help/sites-deploying/ui-recommendations.md).

>[!NOTE]
>
>이 페이지에서는 클래식 UI의 기능 패리티만 다룹니다. 클래식 UI에 없는 터치 지원 UI에 추가되고 고유한 기능은 나열되지 않습니다.

>[!NOTE]
>
>이 목록은 완성을 위해 노력하고 있지만 완벽하지는 않다.

## 범례 {#legend}

* **완료**: 이 기능은 터치 지원 UI에서 모두 사용할 수 있습니다.
* **대부분**: 이 기능은 대부분 터치 지원 UI에서 사용할 수 있습니다.
* **누락**: 터치 지원 UI에 이 기능이 없으면 클래식 UI를 사용하여 이 작업을 수행해야 합니다.
* **대체됨**: 이 기능은 다르게 작동하는 새 구현으로 대체되었습니다.
* **제거됨**: 이 기능은 터치 지원 UI에 더 이상 존재하지 않으며 대체되지 않습니다.
