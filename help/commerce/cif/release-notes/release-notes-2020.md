# Commerce Integration Framework GitHub 릴리스 개요

## 릴리스 날짜: 2020년 11월

| GitHub | 버전 | 자세한 릴리스 노트 |
|:-------|:-----:|---------------------:|
| CIF 커넥터 | 1.6.0 | [릴리스 정보](https://github.com/adobe/commerce-cif-connector/releases) |
| CIF 코어 구성 요소 | 1.6.0 | [릴리스 정보](https://github.com/adobe/aem-core-cif-components/releases) |
| CIF Venia 참조 사이트 | 2020.12.01 | [릴리스 정보](https://github.com/adobe/aem-cif-guides-venia/releases) |

### 새로운 기능 {#what-is-new-november}

* 특정 카테고리 페이지에 템플릿 상속이 추가되었습니다. 이 기능을 사용하면 모든 하위 카테고리가 특정 최상위 카테고리에 대해 작성된 템플릿을 상속할 수 있으므로 비즈니스 사용자 효율성을 높일 수 있습니다.

* Venia 참조 storefront가 바닥글에 경험 조각을 사용하도록 업데이트되었습니다. 비즈니스 사용자는 AEM 작성 도구를 사용하여 페이지 바닥글을 편집할 수 있습니다.

### 개선 사항 {#what-is-improved-november}

* 결제 구성 요소가 개선되어 구매자는 미국 외부의 청구/배송 주소를 허용할 대상 국가를 입력할 수 있습니다.

* Adobe 클라이언트 데이터 레이어를 하이드레이트로 확장되는 탐색 구성 요소입니다.

* 여러 버그 수정.

### 새로운 기능 {#what-is-new-october}

* 비즈니스 사용자가 이 구성 요소를 AEM 컨텐츠 페이지에 끌어다 놓아 상거래 데이터로 컨텐츠 페이지를 보강할 수 있도록 해주는 새 카테고리 회전 메뉴 구성 요소가 추가되었습니다.

* 상거래 데이터를 전송하여 Adobe 클라이언트 데이터 레이어를 하이드레이션하도록 확장된 CIF 코어 구성 요소입니다. Adobe 클라이언트 데이터 계층은 데이터를 수집하고 데이터를 디지털 분석 및 보고 서버에 통신하는 표준화된 방법입니다. 자세한 내용은 [Adobe 클라이언트 데이터 레이어](https://github.com/adobe/adobe-client-data-layer/wiki).

* Adobe Commerce 관리자 UI 내에서 구성된 SEO 메타데이터(예: 제목, 메타 설명, 메타 키워드)를 자동으로 채우도록 확장된 제품 세부 사항 및 제품 목록 페이지입니다

* 상거래 티저 구성 요소 버그가 수정되었습니다.

| CIF 원형 | 0.11.0 | [릴리스 정보](https://github.com/adobe/aem-cif-project-archetype/releases) |

### 새로운 기능 {#what-is-new-june}

Adobe Experience Manager에서 지원되는 CIF 코어 구성 요소의 첫 번째 버전입니다.

* 관련성, 가격 및 제품 이름을 기준으로 정렬할 수 있도록 제품 목록 페이지 및 검색 결과 페이지에 제품 정렬을 추가했습니다.

* 카테고리 필터링을 패싯으로 추가하여 구매자가 카테고리를 기반으로 필터링할 수 있습니다.

* ACL을 직접 조작하지 않고 서비스 사용자를 통해 /conf에 액세스할 수 있도록 하기 위한 보안 요구 사항의 일부로서 서비스 사용자 매핑을 추가했습니다. 이제 CIF 코어 구성 요소 가 서비스 사용자를 사용하여 구성에 액세스해야 합니다.

### 개선 사항 {#what-is-improved-june}

* 제품 목록 페이지 및 검색 결과 페이지에 총 항목 수가 표시됩니다. 쇼핑객이 필터를 적용하면 항목 수가 업데이트됩니다.

* 카테고리 쿼리를 제품 검색 쿼리와 결합하여 면처리 검색이 최적화되었습니다.

* 페이지 미리 보기 용 카테고리/제품 선택기 cq:catalogPath

* 여러 버그 수정.
