# Commerce Integration Framework GitHub 릴리스 개요

## 릴리스 날짜:2020년 11월

| GitHub | 버전 | 자세한 릴리스 노트 |
|:-------|:-----:|---------------------:|
| CIF 커넥터 | 1.6.0 | [릴리스 정보](https://github.com/adobe/commerce-cif-connector/releases) |
| CIF 핵심 구성 요소 | 1.6.0 | [릴리스 정보](https://github.com/adobe/aem-core-cif-components/releases) |
| CIF Venia 참조 사이트 | 2020.12.01 | [릴리스 정보](https://github.com/adobe/aem-cif-guides-venia/releases) |

### 새로운 기능 {#what-is-new-november}

* 템플릿 상속이 특정 카테고리 페이지에 추가되었습니다. 이 기능을 사용하면 모든 하위 카테고리가 특정 상위 카테고리에 대해 작성된 템플릿을 상속할 수 있으므로 비즈니스 사용자 효율성이 향상됩니다.

* 바닥글에 대해 경험 조각을 사용하도록 공급업체 참조 스토어프런트 업데이트가 업데이트되었습니다. 비즈니스 사용자는 AEM 제작 도구를 사용하여 페이지 바닥글을 편집할 수 있습니다.

### 개선 사항 {#what-is-improved-november}

* 결제 구성 요소가 개선되어 구매자에게 미국 외부에서 청구/배송 주소를 허용하도록 대상 국가를 입력할 수 있습니다.

* Adobe 클라이언트 데이터 레이어에 수분을 제공하도록 확장되는 탐색 구성 요소입니다.

* 여러 버그 수정.

### 새로운 기능 {#what-is-new-october}

* 비즈니스 사용자가 AEM 컨텐츠 페이지에 이 구성 요소를 드래그 앤 드롭하여 커머스 데이터로 컨텐츠 페이지를 풍성하게 만들 수 있도록 해주는 새로운 카테고리 슬라이드 구성 요소가 추가되었습니다.

* 상거래 데이터를 전송하여 Adobe 클라이언트 데이터 레이어에 수분을 제공하도록 확장되는 CIF 핵심 구성 요소입니다. Adobe 클라이언트 데이터 레이어는 데이터를 수집하고 데이터를 디지털 분석 및 보고 서버에 전달하는 표준화된 방법입니다. 자세한 내용은 [Adobe 클라이언트 데이터 레이어](https://github.com/adobe/adobe-client-data-layer/wiki)를 참조하십시오.

* Magento 관리 UI 내에서 구성된 SEO 메타데이터(예: 제목, 메타 설명, 메타 키워드)를 자동으로 채우도록 확장된 제품 세부 사항 및 제품 목록 페이지

* 상거래 티저 구성 요소 버그가 수정되었습니다.

* 개발자는 SEO 우수 사례를 따르도록 제품 슬라이드, 관련 제품 및 주요 카테고리 목록 구성 요소에 대한 HTML 태그를 구성할 수 있습니다.

### 새로운 기능 {#what-is-new-june}

Adobe Experience Manager에서 지원되는 첫 번째 버전의 CIF 핵심 구성 요소입니다.

* 제품 목록 페이지 및 검색 결과 페이지에서 제품 정렬을 추가하여 쇼핑객들이 관련성, 가격 및 제품 이름을 기준으로 정렬할 수 있습니다.

* 카테고리 필터링을 패싯으로 추가하여 구매자가 카테고리를 기반으로 필터링할 수 있습니다.

* ACL을 직접 조작하지 않고 서비스 사용자를 통해 /conf에 대한 액세스를 보장하기 위한 보안 요구 사항의 일부로 서비스 사용자 매핑을 추가했습니다. 이제 CIF 핵심 구성 요소를 사용하여 구성에 액세스해야 합니다.

### 개선 사항 {#what-is-improved-june}

* 제품 목록 페이지 및 검색 결과 페이지에는 총 항목 수가 표시됩니다. 구매자가 필터를 적용하면 항목 수가 업데이트됩니다.

* 카테고리 쿼리를 제품 검색 쿼리와 결합하여 패싯된 검색을 최적화할 수 있습니다.

* 페이지 미리 보기의 카테고리/제품 선택기에서 cq:catalogPath를 확인합니다.

* 여러 버그 수정.
